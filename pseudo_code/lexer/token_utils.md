# token_utils.c 疑似コード

## 関数: new_token
- **概要**:
  - 指定されたタイプと値を持つ新しいトークンを作成する。
- **引数**:
  - `type`: トークンの種類
  - `value`: トークンの値（文字列）
- **処理**:
  1. `t_token` 構造体のメモリを確保する。
  2. 確保に失敗した場合、`NULL` を返す。
  3. `type`、`value` を設定する。
  4. `quote_type` を `0` に初期化する。
  5. `next` を `NULL` に初期化する。
  6. 作成したトークンを返す。

## 関数: add_token
- **概要**:
  - トークンリストの末尾に新しいトークンを追加する。
- **引数**:
  - `tokens`: トークンリストへのポインタ
  - `new`: 追加する新しいトークン
- **処理**:
  1. `new` が `NULL` の場合、何もしない。
  2. リスト `*tokens` が空の場合、`*tokens` を `new` に設定して終了する。
  3. リストの末尾まで走査する。
  4. 末尾のノードの `next` に `new` を連結する。

## 関数: free_tokens
- **概要**:
  - トークンリストとその値のメモリを解放する。
- **引数**:
  - `tokens`: 解放するトークンリスト
- **処理**:
  1. リスト `tokens` を走査するループ:
     - 現在のノードを `tmp` に保存する。
     - 次のノードへ進む。
     - `tmp` の `value` があれば解放する。
     - `tmp` 自体を解放する。
