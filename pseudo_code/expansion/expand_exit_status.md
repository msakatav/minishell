# expand_exit_status.c 疑似コード

## 関数: expand_exit_status
- **引数**:
  - `str`: 展開対象の文字列
  - `exit_status`: 終了ステータス
- **処理**:
  1. 結果文字列 `result` を空文字で初期化する。
  2. 文字列 `str` を走査するループ:
     - `$?` が出現するまでの部分文字列を特定する。
     - その部分文字列を `result` に結合する。
     - `$?` が見つかった場合:
       - `exit_status` を文字列に変換する。
       - 変換した文字列を `result` に結合する。
       - インデックスを2つ進める。
  3. `result` を返す。
