# expand_exit_status.c 疑似コード

## 関数: append_exit_status
- **概要**:
  - 終了ステータスを文字列に変換し、結果文字列に追加する。
- **引数**:
  - `result`: 結果文字列へのポインタ
  - `exit_status`: 終了ステータス
  - `i`: 現在のインデックスへのポインタ
- **処理**:
  1. `exit_status` を文字列に変換する。
  2. `result` にステータス文字列を結合する。
  3. インデックス `i` を2進める（`$?` の分）。

## 関数: expand_exit_status
- **概要**:
  - 文字列内の `$?` を終了ステータスの値に置換する。
- **引数**:
  - `str`: 入力文字列
  - `exit_status`: 終了ステータス
- **処理**:
  1. 結果文字列 `result` を空文字で初期化する。
  2. 入力文字列を走査するループ:
     - `$?` が見つかるまでの部分文字列を抽出し、`result` に結合する。
     - `$?` が見つかった場合:
       - `append_exit_status` を呼び出し、終了ステータスを展開して結合する。
  3. 結果文字列を返す。
