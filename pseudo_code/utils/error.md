# error.c 疑似コード

## 関数: print_error
- **概要**:
  - 標準エラー出力にエラーメッセージを表示する。
- **引数**:
  - `cmd`: コマンド名（省略可）
  - `msg`: エラーメッセージ
- **処理**:
  1. 標準エラー出力に "minishell: " を出力する。
  2. `cmd` が指定されていれば、`cmd` と ": " を出力する。
  3. `msg` を出力し、改行する。

## 関数: exit_error
- **概要**:
  - エラーメッセージを表示してプログラムを終了する。
- **引数**:
  - `msg`: エラーメッセージ
  - `exit_code`: 終了コード
- **処理**:
  1. `print_error(NULL, msg)` を呼び出し、エラーメッセージを出力する。
  2. 指定された `exit_code` で終了 (`exit`) する。
