# builtin_echo.c 疑似コード

## 関数: is_n_flag
- **引数**:
  - `arg`: チェックする引数文字列
- **処理**:
  1. `arg` が `NULL`、または `-` で始まらない、または2文字目が `n` でない場合、`0` を返す。
  2. 2文字目以降をループで確認する:
     - `n` 以外の文字が含まれている場合、`0` を返す。
  3. すべて `n` であれば `1` を返す。

## 関数: builtin_echo
- **引数**:
  - `args`: コマンド引数の配列
- **処理**:
  1. 改行フラグ `newline` を `1` (有効) に初期化する。
  2. 引数のインデックス `i` を `1` から開始する。
  3. `args[i]` が存在し、かつ `is_n_flag` が真である間ループする:
     - 改行フラグ `newline` を `0` (無効) に設定する。
     - `i` をインクリメントする。
  4. 残りの引数についてループする:
     - `args[i]` を標準出力に出力する。
     - 次の引数 `args[i + 1]` が存在する場合、スペースを出力する。
     - `i` をインクリメントする。
  5. 改行フラグ `newline` が有効な場合、改行を出力する。
  6. `0` を返す。
