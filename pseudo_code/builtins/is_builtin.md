# is_builtin.c 疑似コード

## 関数: is_builtin
- **概要**:
  - 指定されたコマンドがビルトインコマンドかどうかを判定する。
- **引数**:
  - `cmd`: コマンド名文字列
- **処理**:
  1. `cmd` が `NULL` の場合、`0` を返す。
  2. コマンド名が以下のいずれかと一致する場合、`1` を返す:
     - "echo"
     - "cd"
     - "pwd"
     - "export"
     - "unset"
     - "env"
     - "exit"
  3. 一致しない場合、`0` を返す。
